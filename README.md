# Women Health Agentic AI

Multi-agent AI system for women's health symptom analysis using local LLM (Ollama) and Gradio UI.

## Features

- Symptom extraction
- Disease hypothesis
- Medical reasoning
- Risk validation
- Severity classification
- Patient-safe moderation
- Gradio chatbot interface

## Tech Stack

- Python
- Ollama (LLaMA)
- Gradio
- Multi-Agent Architecture

## Run

```bash
ollama serve
python app.py
